# Bridge Schema Bundle

This directory contains JSON Schema contracts for `xptool-bridge/1` messages.

- `auth_hello.schema.json`
- `auth_proof.schema.json`
- `auth_session.schema.json`
- `signed_command_envelope.schema.json`
- `command_ack.schema.json`
- `bridge_health.schema.json`
- `snapshot_dto.schema.json`
- `interaction_command.schema.json`
- `interaction_result.schema.json`
- `runtime_state.schema.json`
